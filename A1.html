My Latest Project: AI-Powered Task Manager

Posted on August 30, 2024 by Vincent Chiang

In this project, I developed an AI-powered task manager using SwiftUI and CoreML. Here's a snippet of the main view:


struct ContentView: View {
    @State private var tasks: [Task] = []
    @State private var newTaskTitle: String = ""

    var body: some View {
        NavigationView {
            List {
                ForEach(tasks) { task in
                    TaskRow(task: task)
                }
                .onDelete(perform: deleteTask)

                HStack {
                    TextField("New task", text: $newTaskTitle)
                    Button(action: addTask) {
                        Image(systemName: "plus.circle.fill")
                    }
                }
            }
            .navigationTitle("AI Task Manager")
        }
    }

    func addTask() {
        let newTask = Task(title: newTaskTitle)
        tasks.append(newTask)
        newTaskTitle = ""
    }

    func deleteTask(at offsets: IndexSet) {
        tasks.remove(atOffsets: offsets)
    }
}

This code creates a simple interface for adding and removing tasks. The AI component (not shown here) helps in prioritizing and categorizing tasks based on user behavior.
